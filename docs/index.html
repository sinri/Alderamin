Alderamin A MySQL Report Generate Solution with PHP

If you use PHP and you do not want to switch your technique stack for report system;

If you do not want to use Stored Procedures to deal complex data work flow but want to keep SQL simple;

If you do not want to be disturbed by the hanging of schedule framework in various language;

If you just want to provide report service and do not to pay too much effort to design, develop and maintain on it;

And so on. Then Alderamin is a solution for you.

Alderamin is provided as a library to carry your report generating processes.
